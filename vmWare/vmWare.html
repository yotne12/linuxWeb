Installing VmWare Tools to Linux

. For Linux distributions, use the integrated open-vm-tools version.

1. First apt-get update

2. apt-get install open-vm-tools

3. Result repository not found

4. Use [sudo dnf upgrade]...it took a long time to upgrade

5. Next you search with [dnf open-vm-tools] for what you want.

VmWare tools install from the command line in linux

Requirement:s

1. Virtual Machine has to be powered on.

2. Guest operating system has to be running.

3. Check that perl is installed.

cd ~, perl -v or run a perl script

nano hello.pl, next perl hello.pl

4. [mount] command and look for /dev/cdrom on /mnt/cdrom type iso9660 (ro, nosuid, nodev)

5. If it's not mounted then make the directory [mkdir /mnt/cdrom]

it does not verify but returns you to the command line.

6. I learn something about mounting CDROM from the command line
